# Proyecto-IA-Robot-Movil-controlado-por-vision
El Robot Móvil 2.0 es un sistema de tracción diferencial controlado por visión artificial. Usando una ESP32 y motores con encoder, el robot primero autoriza al usuario vía reconocimiento facial. El control se ejerce mediante comandos gestuales. La PC procesa la visión y envía instrucciones a la ESP32 por Bluetooth para la ejecución del movimiento.
